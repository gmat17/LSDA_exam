# LSDA exam
This is the GitHub repository of the Large Scale Data Architecture course's exam. Here you can find the required markdown files.

## Business use case
The business use case for this exam is to analyse Formula 1 data for an imanginary betting company which would like to optimse its betting strategies based on historical data. The company wants to identify patterns and trends during the races, which can help them make informed decisions when setting odds and accepting bets from customers, even in case of live betting.

## Data pipeline
- ✅ FastF1: telemetry data of Formula 1 cars
- ✅ OpenWeatherMap API: weather data during the races
